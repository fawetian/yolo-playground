# 02 - OpenCV 进阶 🔬

## 本模块内容

深入学习图像处理的核心技术，为目标检测做准备。

## 子目录

| 目录 | 内容 | 预计时间 |
|-----|------|---------|
| `01_filters/` | 图像滤波（模糊/平滑） | 2 天 |
| `02_morphology/` | 形态学操作 | 2 天 |
| `03_edge_detection/` | 边缘检测 | 2 天 |
| `04_contours/` | 轮廓检测与分析 | 2 天 |
| `05_feature_detection/` | 特征检测与匹配 | 3 天 |

## 学习目标

完成本模块后，你将掌握：

- [ ] 使用各种滤波器进行图像平滑和去噪
- [ ] 理解并应用形态学操作（腐蚀、膨胀等）
- [ ] 使用 Canny、Sobel 等进行边缘检测
- [ ] 检测和分析图像轮廓
- [ ] 使用 SIFT、ORB 等进行特征检测和匹配

## 这些技术的应用场景

| 技术 | 应用场景 |
|-----|---------|
| 滤波 | 图像去噪、预处理 |
| 形态学 | 图像分割、去除噪点 |
| 边缘检测 | 物体边界识别 |
| 轮廓检测 | 物体计数、形状分析 |
| 特征检测 | 图像匹配、目标跟踪 |

## 运行示例

```bash
conda activate yolo
python 01_filters/01_blur_filters.py
python 03_edge_detection/01_edge_detection.py
```
